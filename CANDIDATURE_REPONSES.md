### **Projet 1 : ChessMaster**
*Application Mobile Full-Stack de Gestion de Parties d'Échecs*

---

**📱 Nom du projet et brève description**

**ChessMaster** - Application mobile multifonctionnelle développée en React Native permettant de jouer aux échecs contre une IA, résoudre des puzzles tactiques, apprendre des ouvertures théoriques, suivre ses statistiques de progression et consulter des leaderboards globaux. Architecture complète mobile + backend + gateway + dashboard administratif.

**Technologies** : React Native (Expo), FastAPI (Python), Node.js, Flutter, SQLite, JWT, WebSocket

---

**👨‍💻 Ma contribution spécifique**

**Rôle : Développeur Full-Stack Mobile (100% autonome)**

**Réalisations :**

✅ **Frontend Mobile (React Native)** :
- Développement de 16 écrans interactifs avec navigation hiérarchique (React Navigation)
- Système d'authentification complet (login/register/logout) avec persistance JWT
- Interfaces de catalogues dynamiques (50+ puzzles, 40+ ouvertures, 5 difficultés)
- Gestion d'état avec Context API et AsyncStorage pour offline-first
- UI/UX moderne avec animations (react-native-reanimated), LinearGradient, @expo/vector-icons
- Export de données multi-formats (CSV, PDF, PGN)
- Formulaires avec validation en temps réel

✅ **Backend API (FastAPI - Python)** :
- Architecture REST avec 40+ endpoints documentés (Swagger)
- Authentification JWT et autorisation basée sur les rôles
- 10 modèles SQLAlchemy (Users, Games, Puzzles, Achievements, Leaderboard, etc.)
- Intégration moteur d'IA python-chess avec 5 niveaux de difficulté (ELO 800-2400)
- Système de révision de parties avec calcul d'accuracy
- Génération automatique de PDF avec reportlab

✅ **Gateway (Node.js - Express)** :
- Proxy API centralisé pour sécuriser les appels backend
- Gestion WebSocket pour notifications temps réel
- Health checks et monitoring

✅ **Dashboard Admin (Flutter/Dart)** :
- Panel web d'administration avec statistiques globales
- Gestion CRUD des utilisateurs, puzzles, leaderboards
- Provider pour state management réactif

**Compétences acquises applicables à MyEPI :**
- Workflows multi-niveaux (workflow hiérarchique EPI à 4 niveaux)  
- Catalogues interactifs avec filtres et recherche (catalogue 40+ catégories EPI)  
- Gestion de formulaires complexes et validation  
- Export de données professionnels  
- Architecture scalable client-serveur
